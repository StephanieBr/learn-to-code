Learn to Code

LearnToCode.social is an online community for people learning to develop web and mobile technologies.

We encourage people down a very specific path to get into this industry. This will take significant effort over a long period of time, but if you follow these steps we believe you can be successful creating web and mobile applications that improve the quality of life in the world and earning a liveliehood.

The path begins with studying web and mobile development online. Expect to spend one year taking online classes and building small projects.

If you can follow these steps and demonstrate an ability to build an application we will recommend you for CodeForIndia Fellowships and introduce you to growing startups.

All you need to do is:

  • Get a badge for any track on HackerRank or complete the beginner track on HackerEarth.
  • Develop a web / mobile application using GitHub that interacts with the user, communicates with the server, works offline & displays local language.

We want to help you.

While you ultimately need to figure out what mix of learning materials works best for you, we share learning activites from around the web that might be of interest.

We coach active students and maintain an online community where they can get to know and support each other.
