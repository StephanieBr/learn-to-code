Getting Started

Create Accounts :

  • create an account on Code Academy
  • create an account on Stack Overflow
  • create an account on GitHub
  • create an account on HackerRank
  • create an account on HackerEarth

Go Deep on Code Academy. Complete the HTML/CSS class, then make the website class, and then make a website for yourself on neocities.com. Search around for guidance on what makes a good personal website. Begin your journey programming with the JavaScript class.

When you finish the Javascript programming class, go over and try your hand with HackerRank and HackerEarth and try an easy challenge. Try to complete the challenge yourself, but look around the web for help. Continue progressing through these challenges getting support from the internet community and giving back to the community by answering other peopleâ€™s questions and writing about your discoveries.

When you feel comfortable with the basics of programming pick a technology stack you want to get good at. Android / Java, Javascript, Ruby & Python are all good choices. Look around the web (and on our resources page) for videos and tutorials to help you learn to develop web and mobile applications with these tools. Learn git and create an account on GitHub. Build your code portfolio by publishing all of your assignments in a repository on GitHub.

Find a tutorial to help you set up the basics of your showcase app. Get a basic application going that talks to a database. Then add authentication. Then make it work offline. Then add local language text.

When you have finished the beginner track on HackerEarth or received a badge on HackerRank and you have an app up on GitHub that can do all these things, You are ready to present at RSPCT and start your career in software development with a CodeForIndia Fellowship or a Startup Internship.
